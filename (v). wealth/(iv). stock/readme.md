# stock.

## start with the basic truths
i. **stock is stored resources.**  
   it exists now for future use.

ii. **stock is finite.**  
   once used, it decreases.

iii. **stock can be replenished.**  
   inputs rebuild it over time.

iv. **stock creates options.**  
   reserves give flexibility and security.

---

## what is stock?

**stock** = **a stored reserve of resources held for later use.**

it is:

• measurable  
• limited  
• useful  

---

## stock across key life resources

### time
- **stock use** = saved time for important work  
- **stock loss** = wasted slack  

### money
- **stock use** = savings and capital reserves  
- **stock loss** = spending without replacement  

### skills
- **stock use** = accumulated competence  
- **stock loss** = decay without practice  

### knowledge
- **stock use** = stored insight and memory  
- **stock loss** = forgotten lessons  

### tools
- **stock use** = maintained equipment and assets  
- **stock loss** = broken or unused tools  

### relationships
- **stock use** = built trust and goodwill  
- **stock loss** = neglect and friction  

### networks
- **stock use** = established connections  
- **stock loss** = weak or inactive ties  

### energy
- **stock use** = stored capacity through rest  
- **stock loss** = depletion without recovery  

### health
- **stock use** = accumulated resilience  
- **stock loss** = damage without repair  

### capital
- **stock use** = reserves that fund growth  
- **stock loss** = spending down core resources  

---

## a simple example

you keep a buffer:
- savings in cash  
- extra time in your week  
- backup tools  

now problems don’t force panic.

---

## first principles summary

**stock = stored resources ready for use.**  
build it, and options expand.  
spend it, and options shrink.
