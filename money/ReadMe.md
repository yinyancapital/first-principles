# money — first principles

## start with the basic truths
i. **people trade to get what they need.**  
   no one has everything.

ii. **barter is hard.**  
   you may have apples but need shoes.

iii. **a shared token makes trade easier.**  
   that token is money.

iv. **money can be stored and measured.**  
   it helps compare value over time.

---

## what is money (first principles)?

**money** = **a shared tool used to trade value.**

it helps with:

• exchange  
• saving  
• measuring  

---

## money across key life resources

### time
- **money use** = saving time through better choices  
- **money loss** = buying things that waste time  

### skills
- **money use** = paying for learning and practice  
- **money loss** = spending without growth  

### knowledge
- **money use** = gaining better information  
- **money loss** = ignoring what you know  

### tools
- **money use** = buying tools that multiply effort  
- **money loss** = buying tools you don’t use  

### relationships
- **money use** = supporting trust and cooperation  
- **money loss** = conflict over value and fairness  

### networks
- **money use** = access to new opportunities  
- **money loss** = isolation and missed chances  

### energy
- **money use** = protecting focus and momentum  
- **money loss** = stress and burnout  

### health
- **money use** = care, rest, recovery  
- **money loss** = neglect and long-term cost  

---

## a simple example

you earn **$10**:
- save $5  
- spend $5 on tools  

now money becomes capital.

---

## first principles summary

**money = a shared tool to trade value.**  
used well, it becomes capital.  
used poorly, it disappears.
