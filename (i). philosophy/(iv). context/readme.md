# context.

## start with the basic truths
i. **information is incomplete by itself.**  
   facts need surrounding conditions to be interpreted correctly.

ii. **meaning is relational.**  
   the same thing can mean different things in different situations.

iii. **boundaries shape relevance.**  
   time, place, goals, and constraints determine what matters.

iv. **good judgment depends on framing.**  
   decisions improve when the situation is clearly defined.

---

## what is context?

**context** = **the surrounding conditions that give information, events, or actions their meaning.**  
it is the frame that makes interpretation possible.

---

## how context works

1. **set boundaries**  
   define scope: who, where, when, and why.

2. **identify relationships**  
   map how parts influence one another.

3. **interpret signals**  
   evaluate facts within the defined frame.

4. **update as conditions change**  
   revise interpretation when the frame shifts.

---

## why context matters

- it **reduces misinterpretation**  
- it **improves decision quality**  
- it **reveals what is relevant**  
- it **aligns communication** between people  

---

## a simple example

question: *why can the same message sound helpful in one moment and rude in another?*

first principles:
- words are interpreted relative to tone and timing  
- urgency changes what feels appropriate  
- relationship history affects how intent is inferred  

so meaning changes because **context changes the frame of interpretation.**

---

## context summary

**context = conditions + relationships → meaning.**  
change the frame, and interpretation changes.
