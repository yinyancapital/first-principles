# skills — first principles

## start with the basic truths
i. **people can learn and improve.**  
   skills grow through practice.

ii. **practice takes time.**  
   no skill appears instantly.

iii. **better skills create better results.**  
   they solve problems and build value.

iv. **unused skills fade.**  
   without practice, they weaken.

---

## what are skills (first principles)?

**skills** = **learned abilities that let you do things well.**

examples:

• writing  
• building  
• selling  
• coding  
• teaching  

---

## skills across key life resources

### time
- **skill use** = practice, repetition, improvement  
- **skill loss** = neglect and decay  

### money
- **skill use** = earning through work and service  
- **skill loss** = low value and limited options  

### knowledge
- **skill use** = applying what you know  
- **skill loss** = unused understanding  

### tools
- **skill use** = using tools effectively  
- **skill loss** = wasted tools  

### relationships
- **skill use** = communication, trust, teamwork  
- **skill loss** = conflict and misalignment  

### networks
- **skill use** = collaboration and contribution  
- **skill loss** = isolation  

### energy
- **skill use** = focused effort and flow  
- **skill loss** = scattered attention  

### health
- **skill use** = staying strong and capable  
- **skill loss** = decline in ability  

---

## a simple example

you want to play guitar:
- practice 30 minutes  
- play for others  

now skill grows and creates value.

---

## first principles summary

**skills = learned abilities created through time.**  
use them well, and they compound.  
ignore them, and they fade.
