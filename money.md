## money — first principles

### Start with the basic truths
1. **People trade to get what they need.**  
2. **Trading directly (barter) is hard.**  
   You may have apples but need shoes.
3. **A shared token makes trade easier.**  
   That token is **money**.

---

### What is money (first principles)?

**Money = a tool people use to trade value.**

It helps with:
- **Trading** (so you don’t need to barter)
- **Saving** (so value can be stored)
- **Measuring** (so value can be compared)

---

### How money connects to capital

**Capital** is resources used to build.  
**Money** is one of those resources.

So:
- **Money can become capital** when used to build or grow something.

---

### First principles summary

**Money = a shared tool to trade value.**  
**Capital = resources (including money) used to build.**
