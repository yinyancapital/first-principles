# game theory.

## start with the basic truths
i. **outcomes depend on multiple decision-makers.**  
   your result often depends on what others do.

ii. **strategy matters when interests interact.**  
   each player anticipates other players' choices.

iii. **rules shape incentives.**  
   payoff structures change behavior.

iv. **cooperation and conflict can coexist.**  
   players may compete in one area and coordinate in another.

---

## what is game theory?

**game theory** = **the study of strategic decision-making where each actor's best move depends on others' moves.**

---

## game theory breakdown

1. **players**  
   the people, firms, teams, or states making choices.

2. **strategies**  
   available actions each player can take.

3. **payoffs**  
   outcomes each player values (profit, safety, status, etc.).

4. **information**  
   what each player knows about rules, options, and others.

5. **equilibrium**  
   a stable pattern where no player gains by unilaterally changing strategy.

6. **repeated interaction**  
   future consequences can make trust and cooperation rational.

---

## a simple example

two businesses choose price:
- both cut price -> both earn less
- both keep moderate price -> both earn more
- one cuts while one holds -> cutter gains short-term share

game theory shows why **incentives can trap players in worse outcomes**.

---

## game theory summary

**game theory = strategic choice in interdependent situations.**
