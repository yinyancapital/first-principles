### Start with the basic truths
1. **People want to do work faster or better.**  
2. **Using your hands alone is limited.**  
3. **A tool helps you multiply your effort.**

---

### What are tools (first principles)?

**Tools = objects that extend your ability to do work.**

Examples:
- Hammer
- Knife
- Calculator
- Computer

---

### How tools connect to skills

- **Skills tell you how to use tools well.**
- A tool without skill is weak.

So:
- **Skill + tool = stronger results.**

---

### How tools connect to time and money

- Tools can **save time** (faster work).
- Tools can **cost money** but **create more value**.

---

### How tools connect to capital

- Tools are **physical capital** because they help build.
- Better tools can grow capital faster.

---

### How tools connect to knowledge

- Knowledge helps you **choose the right tool**.
- Knowledge helps you **use tools wisely**.

---

### How tools connect to Yin and Yang

- **Yin** = careful choice, maintenance, preparation  
- **Yang** = active use, building, producing  

Both are needed:
- **Yin keeps tools ready.**
- **Yang puts tools to work.**

---

### Simple example

You want to build a shelf:
- **Learn** how to use a saw (knowledge + skill)  
- **Use** the saw to cut wood (tool + action)  

Now you build faster than by hand.

---

### First principles summary

**Tools = objects that multiply your ability to work.**  
**Tools + skills + knowledge = stronger results.**  
**Balance (Yin + Yang) = prepare + use.**
